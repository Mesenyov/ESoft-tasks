# Задачи для Школы программирования E-Soft

В данном репозитории представлены решения задач для поступления в Школу программирования. 
Все задачи разделены по соответствующим директориям для удобства проверки.

## 📂 Структура репозитория

### [Task 1: Визуализация дома (HTML/CSS)](./Task1)
*Создание веб-страницы с изображением дома с использованием исключительно CSS.*

### [Task 2: Глубокое копирование объекта (JS)](./Task2)
*Реализация функции `deepClone(obj)` для рекурсивного глубокого копирования сложных структур данных.*

### [Task 3: Правильная скобочная последовательность (JS)](./Task3)
*Алгоритм для проверки корректности строки, состоящей из скобок `()`, `[]`, `{}`.*
